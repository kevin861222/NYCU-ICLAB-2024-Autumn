### 實驗概述
相較往年是一個非常佛心的 final project，主要就是實現以下四個功能。
1. AXI 讀寫 DRAM 。
2. 瑣碎但簡單的運算（加減、除法、比大小）。
3. 切 pipeline 壓低 Clock Period 。
4. 預判圖片內資訊（後面詳述），減少 DRAM 讀寫需求。


### 實驗評論
1. 這是一個看起來非常簡單，實際上也非常簡單的實驗。
2. 學不太到東西。
3. 這屆學生面試要擔心沒東西放。
